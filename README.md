# My Lib · Personal Library Manager

A modern, production-level library manager built with Vanilla JavaScript, Tailwind CSS, and Firebase.

## Features
- Modular architecture with Vite
- Glassmorphism UI design
- Secure Google Authentication
- Cloud Sync and Real-time updates

## Setup
1. `pnpm install`
2. Create `.env` with Firebase credentials
3. `pnpm dev`
